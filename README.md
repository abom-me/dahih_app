## Features
- [x] Add Task
- [x] Remaining Time for the task
- [x] Add Time Table
- [x] Add Notes
- [x] Add Exams Table
- [x] horizontal and vertical view for the tasks
- [x] Real Time LocalData Storage
- [x] Dark Mode
- [x] Languages (Arabic, English)
- [x] Notifications
- [x] Pomodoro Timer

And a lot more
